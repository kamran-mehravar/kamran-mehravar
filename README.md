### **Multimodal Object Detection on Thermal Images Using Transformer Architecture and Weather Information**

🔗 [Read Thesis](https://etd.adm.unipi.it/t/etd-02062026-161851/)

This work focuses on improving object detection in thermal imagery by integrating environmental data such as **temperature** and **humidity** into transformer-based architectures.  
The goal is to enhance model robustness under **environmental and seasonal domain shifts**.
